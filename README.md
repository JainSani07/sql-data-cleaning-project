# SQL Data Cleaning Project

## Dataset
Raw dataset containing missing values, duplicates and inconsistent formats.

## Objective
To clean and prepare raw data for analysis using SQL.

## Tasks Performed
- Removed duplicate records
- Handled NULL values
- Standardized text and date formats
- Created clean final tables

## Tools Used
- SQL (MySQL / SQLite)

## Outcome
Prepared a clean dataset ready for analysis and visualization.
